## Project Overview

This project aims to build a multimodal fashion outfit recommendation system that suggests compatible clothing items given an image of a fashion product.

For example, if a user inputs an image of a shirt, the system will recommend matching items such as pants, shoes, or jackets.

The system combines visual information from clothing images and textual descriptions of fashion items to learn compatibility relationships.

## Dataset

We plan to use one of the following datasets:

* DeepFashion Dataset
* Polyvore Dataset

These datasets provide clothing images along with textual descriptions and outfit combinations.
